Animatable CSS tooltips

Following on from the CSS tooltips I posted a little while ago, I offer up another solution.

These differ from my original post both in markup and function. The original tooltips made use of the :before and :after pseudo-elements and while this is probably the most succinct method the major downfall is that CSS animations and transitions can not, at this time, be applied to pseudo-elements.

With only a minor change in the markup, these tooltips are completely animatable.

View them in action below or head over to the GitHub repository.

Base markup

The base markup for these tooltips is pretty straightforward. The content for your tooltip goes between the span tags.

<a href="#" class="tip-trigger"><span>Yay Tooltip!</span>hover</a>

Adding animations

The interesting stuff starts when you add CSS3 animations into the mix. This is done by adding another class to the anchor tag, for example:

<a href="#" class="tip-trigger tipFade"><span>Yay Tooltip!</span>hover</a>

Fading…

class="tipFade" - Yay Tooltip!hover over me.

class="tipFadeDown" - Yay Tooltip!hover over me.

class="tipFadeUp" - Yay Tooltip!hover over me.

class="tipFadeLeft" - Yay Tooltip!hover over me.

class="tipFadeRight" - Yay Tooltip!hover over me.

Bouncing…

class="tipBounceDown" - Yay Tooltip!hover over me.

class="tipBounceLeft" - Yay Tooltip!hover over me.

class="tipBounceRight" - Yay Tooltip!hover over me.

class="tipBounceIn" - Yay Tooltip!hover over me.

Zooming…

class="tipZoomIn" - Yay Tooltip!hover over me.

class="tipZoomOut" - Yay Tooltip!hover over me.

Spinning…

class="tipSpinIn" - Yay Tooltip!hover over me.

For a comprehensive CSS3 Animation Library you should check out animate.css by Dan Eden. He's pretty much got it all covered there.
